Biomedical Time Series Annotation Tool

This tool facilitates the annotation of biomedical time series such as the electrocardiogram, the photoplethysmogram, and the arterial blood pressure signal.

Begin by selecting a signal from your computer. The file should be in .csv or .txt format. The file should have exactly one row of numeric values, and each value should be separated by a comma.


Optionally, add some existing annotations. The file should be in .csv or .txt format. The file should have exactly one row of integer values, and each annotation location (in samples) should be separated by a comma.
